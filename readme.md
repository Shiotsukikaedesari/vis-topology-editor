## vis-topology-editor
 可视化拓扑流程编辑器

 ## 快捷键
 * `空格 + 鼠标左键`: 拖动画布
 * `鼠标滚轮`: 滚动画布
 * `alt + 鼠标滚轮`: 缩放画布

 > 后期加入页面帮助

## 功能

* 自定义形状节点
* 自定义链接点
* 自定义连线类型
* 自定义样式
* 可定制化的节点连线主体
* 画布图片导出
